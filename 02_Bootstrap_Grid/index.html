Default Grid Setup

  1. You can control the grid by the 'six-responsive breakpoints'
  2. Containers center and horizontally pad your content
  3. Rows are wrappers for columns -> Each column has horizontal padding (called a gutter) for controlling the space in between them. These 'gutters' can be modified via bootstrap modifiers
  4. Columns are flexible -> There are 12 template columns available per row, so you can create many different combinations of elements that span any number of columns. Column classes indicate the number of template columns to span (e.g, col-4, this column takes up the span of 4). widths are set in percentages so you will allways have the same relative sizing.
  5. Gutters are also responsive and cutomizable -> Gutter classes are available across all breakpoints,with all of the same sizes as our 'margin and padding spacing'. If you want to change horizontal gutters use : .gx-* classes, use vertical gutters: .gy-* classes, and you can use g-0 to remove gutters.

Nesting

To nest your content with the default grid, and a new .row and set of .col-sm-* columns within an existing .col-sm-* column. Nested rows should include a set of columns that add up to 12 or fewer (It's not a requirement that you use all 12 available columns )
